# 零仓储电商（Dropshipping）完整操作手册

## 第一步：选品（1-2天）

### 选品原则
- ✅ 体积小、重量轻（运费低）
- ✅ 不易碎（破损率低）
- ✅ 售后简单（不用修不用换）
- ✅ 利润空间>30%（能覆盖差价+广告费）
- ❌ 大件家具
- ❌ 服装（尺码问题）
- ❌ 食品（保质期）
- ❌ 复杂电子（售后头疼）

### 推荐品类（选1-3个先测）
1. **配饰类**：耳钉、项链、手链、钥匙扣
2. **数码配件**：数据线、充电头、屏幕贴膜、手机支架
3. **家居小件**：收纳盒、挂钩、置物架
4. **美妆个护**：假睫毛、美甲贴、化妆刷

### 选品方法
1. **上目标平台搜**（Amazon/TikTok/Etsy）
   - 搜索关键词："earrings for women"、"phone case"、"desk organizer"
   - 看什么产品评论多（>500条）且评分高（>4.2星）
   - 截图3-5个爆款产品

2. **去1688找同款**
   - 把截图的关键词翻译成中文搜
   - 对比价格：Amazon卖$15，1688卖¥15（约$2）= 利润空间OK
   - 看销量、评价（选月销>1000的）

3. **选出1-3个产品**先测
   - 别贪多，跑通一个再扩

---

## 第二步：找货源并谈代发（1天）

### 操作步骤
1. **在1688选3-5家供应商**
   - 看商家评分：>4.8星
   - 看响应速度：问个问题，多久回
   - 看店铺年限：>3年优先

2. **联系供应商谈代发**
   **复制这段话私信商家：**
   ```
   你好，我是做跨境电商的，看到你家[产品名称]不错，想了解一下：
   1. 是否支持一件代发？
   2. 代发时效多久？
   3. 发什么快递（小包还是专线）？
   4. 售后政策（质量问题怎么办）？
   5. 是否可以打我的标签？
   ```

3. **选最靠谱的2家建立合作**
   - 响应快、服务好的优先
   - 发货时效<7天的优先
   - 支持打你自己标签的（品牌化）

## 第三步：采购样品（3-5天）

### 为什么要买样品
- 验证质量（别只看图片）
- 测发货速度
- 拍摄真实产品图/视频

### 操作步骤
1. **选1-2个产品买样品**
   - 数量：每个买1-2个
   - 运费：选最便宜的（小包就行）

2. **收到样品后**
   - 测质量：和图片一样吗？手感如何？
   - 测尺寸：符合平台描述吗？
   - 拍照/录像：
     - 产品正面、侧面、细节特写
     - 使用场景（戴起来/用起来）
     - 对比图（和同类产品放一起）

3. **如果样品不行**
   - 换供应商
   - 别勉强卖，差评毁店

## 第四步：搭建销售渠道（1-2天）

### 推荐组合方案：TikTok Shop + 独立站（Shopify）

### 方案A：TikTok Shop（新手推荐）
**优势：** 流量大、门槛低、自带带货属性

**注册步骤：**
1. 注册TikTok账号
2. 申请TikTok Shop卖家权限
3. 上传产品：
   - 标题：[产品名] + [核心卖点] + [场景]
     例："Sterling Silver Earrings for Women - Hypoallergenic - Gift for Her"
   - 描述：写清楚材质、尺寸、用途
   - 图片：5-9张（白底图+场景图+细节图）
   - 价格：成本×2-3倍（留广告费空间）
4. 设置运费：
   - 免运费（把成本加到产品价里）
   - 或按重量/地区设置

### 方案B：Shopify独立站
**优势：** 品牌化、自主权高、无平台抽成

**注册步骤：**
1. 注册Shopify账号（14天免费试用）
2. 选择主题（选免费的就行）
3. 安装代发App：
   - DSers（免费，支持1688）
   - CJ Dropshipping
4. 上传产品（同TikTok Shop）
5. 设置支付：PayPal、信用卡

### 方案C：Etsy（适合配饰、手工风）
**优势：** 买家粘性高、平台流量精准

**注册步骤：**
1. 注册Etsy账号
2. 开店设置：
   - 店名：简单好记
   - Banner：用Canva做个简单的
3. 上传产品（同上）
4. 设置运费模板

**建议：**
- 新手：先做TikTok Shop，流量大+易上手
- 有经验后：Shopify + TikTok引流，长期发展

## 第五步：制作产品内容（2-3天）

### 核心原则
- **真实**：别P图太过，买家收到落差大就退货
- **展示使用场景**：让买家想象自己拥有
- **突出卖点**：1-2个点讲清楚，别堆砌

### 视频制作（TikTok必备）

**类型1：开箱视频**
- 镜头：产品拆包特写
- 内容：展示包装、产品细节
- 时长：15-30秒
- BGM：轻快音乐

**类型2：使用场景**
- 镜头：产品在真实场景中
- 内容：展示怎么用、效果如何
- 时长：15-45秒
- 文案："看这个[产品名]太方便了"

**类型3：痛点解决**
- 镜头：先展示问题，再展示你的产品解决
- 内容：
  - "还在为[问题]烦恼吗？"
  - "试试这个[产品名]"
  - 特写展示效果
- 时长：30-60秒

### 图片制作（Shopify/Etsy必备）

**需要5-9张图片：**
1. **白底主图**：产品正面、清晰、突出核心卖点
2. **侧面图**：展示厚度、层次
3. **细节图**：放大1-2个亮点（材质/工艺）
4. **场景图1-3张**：产品在真实使用场景中
5. **对比图**：和同类产品放一起（可选）

**制作工具：**
- 手机拍摄 + Canva修图
- 不用买专业设备，自然光就行

## 第六步：开始引流（持续）

### 免费流量（先做这些，0成本）

#### TikTok引流
**发布频率：** 每天1-3条

**内容策略：**
- 轮播产品：别天天发同个，换着来
- 真实出镜：用手机拍，别用素材网视频
- 热点音乐：用TikTok热门BGM

**发布步骤：**
1. 打开TikTok，拍产品视频
2. 加产品链接（TikTok Shop功能）
3. 写文案：
   - "Find it on TikTok Shop: [产品名]"
   - 或："Link in bio"
4. 加标签：
   - #产品相关标签
   - #场景标签（例：#earringsoftheday）
   - #热门标签（例：#fyp）

#### Instagram引流
**发布频率：** 每天1条

**内容：**
- Reels短视频（复用TikTok视频）
- 静态图片（产品场景图）
- Story（互动：投票、问题）

**操作：**
1. 发布内容
2. 在Bio挂Shopify链接
3. 互动：回评论、点赞同类帖子

#### Pinterest引流
**发布频率：** 每天3-5条

**内容：**
- 制作Pin图（竖版产品场景图）
- 描述写清楚：什么产品、什么用途

**操作：**
1. 创建Pin
2. 链接回你的店铺
3. 加关键词标签

#### SEO优化（所有平台通用）

**标题写法：**
- [产品名] + [材质/特点] + [使用场景] + [目标人群]

例：
- "Sterling Silver Earrings for Women - Hypoallergenic - Minimalist Design - Gift for Her"
- "Phone Case for iPhone 15 - Shockproof - Clear Case - Slim Design"

**描述写法：**
- 第1行：产品是什么、有什么特点
- 第2行：适用场景、材质/尺寸细节
- 第3行：适合什么人群、使用注意

### 付费流量（数据跑通后再投）

#### Facebook Ads（推荐新手）
**为什么推荐：** 精准定位、转化率高

**操作步骤：**
1. 开Facebook Business账户
2. 创建广告系列：
   - 目标：销售（Conversions）
   - 预算：先$5-10/天测试
3. 受众设置：
   - 地区：美国/英国/加拿大（根据产品选）
   - 年龄：18-45岁
   - 兴趣：根据产品选（例：配饰选Fashion & Beauty）
4. 广告素材：
   - 用你最好的产品视频/图片
   - 文案：突出痛点+解决方案
5. 运行3-7天看数据：
   - 点击率（CTR）>1%：继续投
   - 转化率>2%：加预算
   - <1%：换素材

#### Google Shopping
**为什么推荐：** 意图明确（搜了才看到）

**操作步骤：**
1. 开Google Ads账户
2. 上传产品数据到Google Merchant Center
3. 创建购物广告系列
4. 运行7天看数据，优化关键词

#### TikTok Ads
**注意：** 流量大但转化低，适合测试

## 第七步：订单处理（日常）

### 订单流程
```
客户下单 → 你在平台收到订单 → 去1688下单给供应商 → 供应商发货给你 → 你填运单号 → 供应商直接发客户
```

### 具体操作

#### 如果用TikTok Shop
1. 客户在TikTok下单，你收到订单通知
2. 看客户收货地址（复制）
3. 去TikTok Shop后台看订单详情：
   - 产品：[产品名]
   - 数量：[X个]
   - 地址：[客户地址]
4. 打开1688，找到对应供应商
5. 下单给供应商：
   - 选择数量：客户买多少你订多少
   - 填客户地址（别填自己的）
   - 选择物流：问过供应商最便宜时效OK的
   - 备注：
     ```
     请务必在包裹上贴以下面单（或直接发客户）：
     [客户姓名]
     [客户地址]
     [客户电话]
     ```
6. 供应商发货，给你运单号
7. 把运单号填到TikTok Shop后台
8. 完成

#### 如果用Shopify + DSers
1. 客户在Shopify下单
2. DSers自动同步订单
3. DSers自动去1688下单
4. 你只需要确认订单、填运单号

#### 如果是手工操作（所有平台通用）
1. 收到订单
2. 打个Excel表格：
   | 订单号 | 产品 | 数量 | 客户地址 | 运单号 |
   |---------|------|------|----------|--------|
   | #001   | 耳钉 | 2    | 美国XXX   |        |
3. 逐条处理
4. 填完运单号标记"已发货"

### 注意事项
- ⚠️ 别漏单：每天固定时间（例：早上9点、晚上9点）处理
- ⚠️ 地址核对：供应商发货前让客户确认地址（防止发错）
- ⚠️ 运单号及时填：平台有发货时限，超时封店
- ⚠️ 供应商不配合：提前备2-3家，随时换

## 第八步：售后处理（日常）

### 常见售后类型及处理

#### 1. 发错货/少件
**流程：**
```
客户反馈 → 你确认问题 → 联系供应商 → 供应商确认 → 供应商补发 → 你告知客户 → 客户确认收到
```

**话术：**
- 对客户："非常抱歉给您带来不便，我们已经联系供应商确认，会尽快为您补发"
- 对供应商："[订单号]客户反馈发错/少件，请核实并补发，费用你们承担"

#### 2. 质量问题/损坏
**流程：**
```
客户发图 → 你核实问题 → 供应商确认责任 → 供应商补发或退钱 → 你处理客户
```

**责任判断：**
- 供应商发错货、质量差：供应商负责
- 客户人为损坏：你自己承担（或和供应商协商分担）

#### 3. 客户不满意要退
**流程：**
```
客户申请退款 → 你先协商（补发？优惠？）→ 协商不成 → 走平台退款
```

**注意：**
- 供应商不退的你也退不了
- 退款前先和供应商确认

### 售后话术模板

**客户投诉：**
```
您好，非常抱歉给您带来不好的体验。
1. 请您拍摄一下产品照片/视频，我这边核实一下
2. 确认问题后，我会立即联系供应商为您处理
3. 一般3-5个工作日内会有解决方案
```

**催发货：**
```
您好，您的订单已经发货，预计7-15天到达。
国际物流可能受海关、天气等因素影响，请耐心等待。
如有问题随时联系我。
```

### 降低售后率的技巧
1. **产品描述要真实**：别夸大，降低期望落差
2. **选靠谱供应商**：质量稳定，售后少
3. **买样品确认**：质量OK才卖
4. **提前预警物流**：告知客户时效，减少催单

## 第九步：数据分析和优化（每周）

### 每周看哪些数据

#### 销售数据
- 哪个产品卖最多？→ 加大推广
- 哪个产品退货多？→ 下架或换供应商
- 哪个产品利润高？→ 重点推

#### 流量数据
- 哪个平台转化率最高？→ 加预算
- 哪条视频播放量高？→ 复制这种内容
- 哪个广告CTR高？→ 保留优化

#### 成本数据
- 广告成本占销售额多少？→ 控制在30%以内
- 退货运费占多少？→ 找质量更好的供应商
- 利润率多少？→ <20%的产品下架

### 优化方向

1. **爆款继续放大**
   - 加大广告预算
   - 做更多内容
   - 延伸产品线（例：耳钉卖好，加项链）

2. **淘汰品果断下架**
   - 2周卖不动，换产品
   - 别赌爆款，数据说话

3. **供应商优化**
   - 售后差的供应商，果断换
   - 发货慢的，找更快的

## 常见问题FAQ

### Q1: 需要启动资金吗？
A: 理论上0启动，但建议准备$100-500：
- 买样品：$50-100
- 投广告测试：$50-300
- 备用（退换货）：$50-100

### Q2: 什么时候开始投广告？
A: 等数据跑通：
- 产品上传完成
- 内容有3-5条
- 有自然流量但转化低 → 再投广告

### Q3: 供应商直接发给客户，客户知道吗？
A: 不知道，因为：
- 供应商包装很简单（普通纸箱）
- 你要求打你的品牌标签（或者供应商直接贴客户面单）
- 客户只知道是你卖的

### Q4: 退货率高怎么办？
A: 检查：
1. 产品描述是否真实
2. 图片是否过度P图
3. 供应商质量是否稳定
4. 是否选对了目标人群

### Q5: 怎么避免和供应商扯皮？
A:
1. 提前谈清楚售后政策
2. 购买样品确认质量
3. 多备2-3家供应商
4. 小金额自己赔了（比扯皮划算）
